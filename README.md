# HABIT MANAGER
Hello! This is my project based on the Python programming language. The project is called Habit Manager, and as the name suggests, the user can set various tasks of varying difficulty and mark them as completed. So, I used various technologies. The most important one, of course, was Django. This framework connects web pages and Python code. I used Python codes for backend and HTML with CSS for Frontend of the website.
I didn't host the site, so we launch it either through the VS code or through the computer's terminal.
Enter:

cd habit_manager

then:

python manage.py runserver

and copy the link to the site, paste it into the browser, this way you will access the website
And how site's work. The user visits the website. A page prompting account page. If they already have an account, they simply sign in; if not, they create a new account. Next, they're redirected to the main page of the site, where they can create habits. They then create them, give them a name and description, as well as a difficulty level and completion date. They can then mark them as completed.
